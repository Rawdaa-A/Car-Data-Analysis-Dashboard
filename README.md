# Car Data Analysis Dashboard | SQL Normalization & Excel Visualization 🚗📊

## 📝 Project Overview
This project demonstrates an end-to-end data pipeline, from raw data processing in **SQL** to advanced visualization in **Excel**. I transformed a flat `Car_Data.csv` file into a fully normalized relational database and visualized the insights through an interactive dashboard.

## 🛠️ Technical Workflow (Data Engineering & Analysis)

1. **Database Setup & ETL:** Imported raw data into **SQL Server/PostgreSQL** and created a primary staging table.
2. **Data Normalization:** Structured the data into normalized entities to reduce redundancy:
   * **Brands Table**
   * **Models Table**
   * **Owners Table**
   * **Transactions Table**
3. **Advanced SQL Techniques:** Since I had "Read-Only" permissions, I utilized **CTEs (Common Table Expressions)** to generate IDs and establish relationships dynamically.
4. **Live Data Connection:** Connected the SQL Database to **Excel via ODBC**, using **Advanced Queries** to fetch structured, clean data for analysis.

## 📈 Dashboard Insights
The final Excel Dashboard provides a deep dive into:
* **Brand Performance:** Analysis of car sales and market share by brand.
* **Pricing Trends:** Tracking price fluctuations and valuation.
* **Transaction History:** Insights into sales volume and owner trends.
